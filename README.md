# SportZone Product Request System

A screenshot-friendly React sample application for the full-stack e-commerce product request assignment.

## Purpose
This sample shows the assignment workflow using a sports product system:

- User registration and login
- User profile editing with profile image preview
- Product listing and product details
- Add to cart
- Add to wishlist
- Request product from admin
- Admin dashboard
- Admin product creation and product management
- Admin request accept/reject workflow
- Admin user ban/unban workflow

## Technology

- React.js
- Vite
- LocalStorage data storage
- CSS responsive design

This is a frontend-only reference sample using localStorage. Students can later connect a Node.js backend and MongoDB/MySQL database.
